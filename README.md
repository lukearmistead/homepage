# armistead.xyz

Personal website. Hand-rolled HTML + CSS, no frameworks, no build step.

Hosted via GitHub Pages from the `main` branch with custom domain `armistead.xyz`.

## Custom domain setup

1. `CNAME` file in repo root points to `armistead.xyz`
2. DNS A records point to GitHub Pages IPs
3. CNAME record: `www` → `armistead.github.io`
